# Intermediate Phase Classroom Management Assistant

## Project Overview

The Intermediate Phase Classroom Management Assistant is an AI-powered productivity platform designed to help Intermediate Phase teachers (Grades 4–6), student teachers, Heads of Department (HODs), and school administrators automate everyday administrative tasks.

Teachers often spend valuable time drafting emails, summarizing meeting notes, planning schedules, and documenting learner behaviour. This application leverages Artificial Intelligence to streamline these repetitive tasks, improve productivity, and allow educators to focus more on teaching and learner support.

The platform combines multiple AI-powered tools into one integrated dashboard, providing a simple, efficient, and professional solution for classroom management and communication.

---

## Problem Statement

Intermediate Phase teachers face increasing administrative workloads that reduce the time available for lesson preparation, learner engagement, and academic support. Common tasks such as writing emails, organizing schedules, attending meetings, and recording learner behaviour can be time-consuming.

This project addresses these challenges by developing an AI-powered assistant that automates administrative processes and improves workplace productivity within the education sector.

## System Architecture

User Input → AI Prompt Processing → OpenAI API → AI Response → User Dashboard Output

The system accepts teacher input, processes it through carefully engineered prompts, generates AI responses, and displays structured outputs within the application dashboard.

---

## Prompt Engineering Strategy

The application uses structured prompts that:

* Define the user role
* Specify the task objective
* Provide context
* Define output format
* Specify tone requirements
* Include validation instructions

This ensures consistent, professional, and accurate outputs.

* Promoting transparency

## Setup Instructions

### Prerequisites

Before running the project, ensure you have:

* Node.js installed
* npm installed
* OpenAI API key
* Internet connection

### Installation

#### Step 1: Clone the Repository

```bash
git clone https://github.com/yourusername/intermediate-phase-classroom-management-assistant.git
```

#### Step 2: Navigate to the Project Folder

```bash
cd intermediate-phase-classroom-management-assistant
```

#### Step 3: Install Dependencies

```bash
npm install
```

#### Step 4: Configure Environment Variables

Create a `.env` file in the project root:

```env
OPENAI_API_KEY=your_openai_api_key
```

#### Step 5: Start the Development Server

```bash
npm run dev
```

#### Step 6: Open the Application

```text
http://localhost:5173
```
